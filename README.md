# customer-segmentation-analysis
Customer Segmentation and Behavioral Targeting Analysis using Pytho, MySQL and Power BI
Project Overview

This project focuses on analyzing customer purchasing behavior using the Online Retail dataset. The goal is to segment customers based on Recency, Frequency, and Monetary (RFM) analysis and derive actionable business insights for targeted marketing strategies.

Tools & Technologies
Python (Pandas, Matplotlib)
MySQL
Power BI
Excel (data handling)

Workflow
1. Data Cleaning (Python)
Removed duplicates
Handled missing values
Created revenue column

3. KPI Analysis
Total revenue
Top customers

Country-wise sales

4. RFM Analysis
Recency (how recently a customer purchased)
Frequency (how often they purchase)
Monetary (how much they spend)
5. Customer Segmentation

Segments created:

Champions
Loyal Customers

Lost Customers

Revenue by segment

7. Power BI Dashboard
Customer segmentation view

Improve customer retention
